*** Tests the DIFFERENCE between:
    - adjusting the alpha after each encounter
    - adjusting the alpha only after a session has ended



*** NOTE ***
Absolute values should obviously not be taken seriously, but just as a comparison.



*** DESCRIPTION ***
- Takes the average values of N LEARNING sessions in which alphas are adjusted after each encounter with an item
- Does the same, but adjusts the alphas only after the session is over
- Compares the average values.
- NOTE: the actual alphas of words stay the same between ALL LEARNING sessions and between ALL TESTS
- NOTE: the activations stored for each encounter do NOT get updated after learning, they stay the same
